# Documentation

## Methodology

- [`methodology.md`](methodology.md) — conceptual overview of the technology scouting practice.

## Assessment framework

- [`assessment-criteria.md`](assessment-criteria.md) — reference for the six scoring criteria, their scales, and how to choose between values.
- [`assessment-workflow.md`](assessment-workflow.md) — operator walkthrough for assessing a technology end-to-end (capture → score → place → publish).
- [`ring-placement.md`](ring-placement.md) — translating an assessment into a ring recommendation (Invest / Pilot / Explore / Monitor).
- [`assessment-api.md`](assessment-api.md) — API reference for creating factsheet versions and assessments, and for setting the ring.
